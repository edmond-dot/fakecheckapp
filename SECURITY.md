## 🔒 Security Vulnerability Reporting

If you discover a security vulnerability in FakeCheckApp (e.g., a way for users' API keys to be exposed, XSS issues, etc.), please report it responsibly:

### How to Report

1. **Do NOT** create a public GitHub issue
2. **Do NOT** publicly disclose the vulnerability
3. Open a **private security advisory** via GitHub: Repository → Security tab → "Report a vulnerability"
4. Include:
   - Description of the vulnerability
   - Steps to reproduce
   - Potential impact
   - Suggested fix (if known)

**Edmond Azbel** will respond within 7 days and credit you in the fix announcement (if you wish).

### What's In Scope

✅ Security issues we want to know about:
- API key leakage paths
- Cross-site scripting (XSS)
- Cross-site request forgery (CSRF)
- Local storage exposure
- Service worker exploits
- Issues that could harm users

❌ Out of scope:
- Cosmetic/UX issues (these are bug reports, not security)
- Third-party API issues (report to Anthropic/OpenAI/Google directly)
- Issues requiring physical access to the user's device

## ©️ Copyright Infringement Reporting

If you discover someone **copying, redistributing, or claiming authorship** of FakeCheckApp without permission from Edmond Azbel, please help by reporting it.

### What Counts as Infringement

- ❌ Republishing this code under a different name or author
- ❌ Removing "Edmond Azbel" attribution from any file
- ❌ Creating a competing app that copies the unique methodology
- ❌ Using the "FakeCheckApp" trademark without authorization
- ❌ Selling, licensing, or distributing this code without permission
- ❌ Stripping watermarks, copyright notices, or license terms
- ❌ Falsely claiming authorship of any portion

### How to Report Infringement

Open an issue in this repository with:
- Title: `[INFRINGEMENT REPORT] - [Brief description]`
- Tag: `infringement`
- Include:
  - URL of the infringing work
  - Description of what was copied
  - Evidence (screenshots, links)
  - Your name (optional, but helpful for follow-up)

**Edmond Azbel** investigates all reports and may pursue legal action including:
- DMCA takedown notices
- Cease-and-desist letters
- Copyright infringement lawsuits
- Trademark infringement lawsuits (once trademark is registered)
